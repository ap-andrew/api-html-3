Maps

/maps
A persistence resource that supports storing arbitrary data
Maps provide an API for storing arbitrary name/value pairs that can be access at runtime by custom policies, or for other custom runtime requirements such as protocol support. Each map has one or more 'entries', each of which which defines a name/value pair that can be fetched, updated, deleted via the API.
