Get Authorization Code

/oauth2/authorizationcodes/{authCode}

Get a specific Authorization Code

API Resource: Oauth 2.0 Authorization Code

Verb: GET

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/oauth2/authorizationcodes/{authCode}
Request Payload: 




Request Payload Elements: 




	
		

			Name
			Description
			Default
			Required?
		

		






Response Payload: 


	
 {
  "attributes" : [ {
    "name" : "region",
    "value" : "south"
  } ],
  "client_id" : "kjvzAT2kR03jcZGQDod6VtMt8Lj0Ce6c",
  "code" : "m9PEEI6I",

  "createdBy" : "Admin",

}




Response Payload Elements:
























	
		

			Name
			Description
		

		


attributes 
Name/value pairs used to extend the default auth code profile




client_id 
The consumer key of the app




code 
The value of the authorization code




createdAt 
Unix time when the auth code was generated




createdBy 
User name of the Apigee user who created the auth code




expiresAt 
Unix time when the auth code expires




issuedAt 
Unix time when the auth code was issued




lastModifiedAt 
Unix time when the auth code was last modified




lastModifiedBy 
User name of the Apigee user who last modified the auth code




scope 
Comma separated list of OAuth scopes




status 
Status of the auth code, which changes to expired when the validity period has passed







Errors 


	
Failure 404 with proper error message
{
  "code" : "keymanagement.service.invalid_request-authorization_code_invalid",
  "message" : "Invalid Authorization Code",
  "contexts" : [ ]
}
