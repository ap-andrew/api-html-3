List Developers in a Company

/companies/{companyname}/developers/

Lists all Developers associated with a Company

API Resource: Company Developers

Verb: GET

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/companies/{companyName}/developers
Response Payload: 
Success 200 with proper response
{
  "developer" : [ {
    "email" : "developer@apigee.com",
    "role" : "developer"
  } ]
}
Response Payload Elements:
Name Description
email Email address of the developer
role Role of the developer within the company--currently unenforced by Apigee
Errors 
Failure 4xx with proper error message
{
 "code" : "developer.service.DeveloperDoesNotExistInTheGivenCompany",
 "message" : "Developer does not exist in the given company",
 "contexts" : [ ]
}
Auth: Basic Auth
