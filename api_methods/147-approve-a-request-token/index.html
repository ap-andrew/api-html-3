Approve a Request Token

/oauth/1/requesttokens/{requesttoken}?action=approve

Approves an OAuth 1.0a Request Token

API Resource: Oauth 1.0a Request Tokens

Verb: POST

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/oauth/1/requesttokens/{requesttoken}?action=approve
Response Payload: 
Success 200 with proper response
Response Payload Elements:
Name Description
Errors 
Failure 404 with proper error message
{
  "code" : "keymanagement.service......",
  "message" : "Request Token doesnot exist",
  "contexts" : [ ]
}

Error Code:
1. keymanagement.service.oauth10_request_token_doesnot_exist
3. keymanagement.service.application_not_approved
4. keymanagement.service.oauth10_request_token_approve_failed
Auth: Basic Auth
