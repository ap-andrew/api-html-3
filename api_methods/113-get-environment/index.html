Get Environment

/environments/{environmentName}

Gets details for an environment

API Resource: Environments

Verb: GET

Returns an environment's details.

cURL: 
curl -u :myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/environments/{environmentName}
Response Payload: 
{
  "createdAt" : 1336660158465,
  "createdBy" : "admin",
  "lastModifiedAt" : 1336660158465,
  "lastModifiedBy" : "admin",
  "name" : "test",
  "properties" : {
    "property" : [ ]
  }
Response Payload Elements:
Name Description
createdAt Unix time when the environment was created
createdBy User name of the Apigee user who created the environment
lastModifiedAt Unix time when the environment was last modified
lastModifiedBy User name of the Apigee user who last modified the environment
name Name of the environment
properties A list of properties reserved for use by Apigee
Errors 




Auth:
Basic Auth
