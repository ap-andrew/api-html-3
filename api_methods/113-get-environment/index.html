Get Environment

/environments/{environmentName}

Gets details for an environment

API Resource: Environments

Verb: GET

Returns an environment's details.

cURL: 
curl -u :myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/environments/{environmentName}
