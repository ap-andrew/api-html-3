Remove a Developer from a Company

/companies/{companyname}/developers/{developerEmail}

Removes the association of a Developer with a Company

API Resource: Company Developers

Verb: DELETE

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/companies/{companyName}/developers/{developerEmail}
