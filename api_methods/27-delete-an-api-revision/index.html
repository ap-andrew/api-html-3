Delete an API Revision

/apis/{apiName}/revision/{revisionNumber}

Deletes a specific revision of an API

API Resource: APIs

Verb: DELETE

Deletes a specific revision of an API. Note that a deployed revision cannot be deleted. You must undeploy the revision before deletion.

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/apis/{apiName}/revisions/{revisionNumber}
