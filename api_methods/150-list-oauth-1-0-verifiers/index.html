List Oauth 1.0 Verifiers

/oauth/1/verifiers

List Oauth 1.0 Verifiers

API Resource: Oauth 1.0a Verifiers

Verb: GET

Gets a list of all verifiers and details for each verifier in an Organization.

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/oauth/1/verifiers

Response Payload Elements:






















	
		

			Name
			Description
		

		


code 
Fetches the detail of the verifier code provided by the resource owner 




appUserId 
Resource owner's user ID




attributes 
Arbitrary name/value elements attached to the profile for the verifier




requestToken 
Token to re-issue the request token




createdAt 
UNIX time the verifier was created




createdBy 
Apigee user account name of the individual who submitted the POST request that created the verifier




expiresAt 
Expiry time period of the token to be generated; token expires/is deleted when the expiry time ends




issuedAt 
UNIX time the verifier code is issued




lastModifiedBy 
Apigee user account name of the individual who submitted the request that most recently modified this entity	




lastModifiedAt 
UNIX time the app was last modified
