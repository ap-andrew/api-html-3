Get Statistics for Top APIs

/environments/{environmentName}/stats/apis

Returns 5 APIs with the highest number of active end users (app end users with throughput) in the last 24 hours.

API Resource: Analytics

Verb: GET

Use the GET method along with the 'topk' parameter to view the first five top ranked apps with the highest number of active end users (app end users with throughput) in the last 24 hours. The graph lists the average daily throughput for these top APIs over the last 30 days for a given customer in a specific organization and environment.

cURL: 
curl -u myname:mypass http://api.enterprise.apigee.com/v1/o/myorg/environments/prod/stats/apis?select=message_count&timeRange=7%2F1%2F2012%2000%3A00~8%2F1%2F2012%2000%3A00&timeUnit=hour&sortby=message_count&sort=DESC&topk=5
