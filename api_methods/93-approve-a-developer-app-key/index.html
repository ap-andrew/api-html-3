Approve a Developer App Key

/developers/{developerEmal}/apps/{appName}/keys/{consumerKey}

Changes the status of a consumer key from 'pending' to 'approved'

API Resource: Developer App Keys

Verb: POST

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/developers/{developerEmail}/apps/{appName}/keys/{consumerkey}?action=approve
Response Payload: 
Success 204 with no response
