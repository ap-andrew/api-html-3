Gets the count of API Resources for a Developer App in an Organization

/developers/{developerEmail}/apps/{appname}?query=count&entity=apiresources

Gets a count of all API Resources (URIs) in the API Products accessible by an App

API Resource: Developer Apps

Verb: GET

Returns a count of all API Resources (URIs) available to be access by an app through the API products the app is associated with.

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/{developerEmail}/apps/{appname}?query=count&entity=apiresources
Response Payload: 
{
  "value" : 3
}
