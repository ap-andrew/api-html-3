Create an API

/apis

Creates an API facade (and an API proxy configuration) on Apigee

API Resource: APIs

Verb: POST

This method creates the framework of an API proxy in Apigee, providing an alternative to creating APIs via the UI. The API will not be accessible at runtime until a proxy endpoint and target endpoint are defined. Usually, this request is followed up with an export API proxy method, which brings the API proxy bundle to the developer's local machine, where AP proxy can be configured and then imported back into Apigee for execution.

cURL: 
curl -u myname:mypass https://api.enterprise.apigee.com/v1/o/{myorg}/apis
